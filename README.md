# PTA-Next (pta-next)

PTA-Next

## Building

### Requirements

Only Windows is supported

- [Node.js 12](https://nodejs.org/en/)
- [node-gyp dependencies](https://github.com/nodejs/node-gyp#installation) (your own copy of Visual Studio and Python OR windows-build-tools)

### Instructions

```bash
yarn
yarn build
```

### Dev build

```bash
yarn rebuild
yarn dev
```
